科幻电影中的记忆

1.

《她的回忆》是大友克洋担任总监督的《回忆三部曲（Memories）》这一经典动画电影中的第一个短片。大友克洋在日本动漫界影响力巨大，深受漫迷的喜爱与推崇。《回忆三部曲》有浓烈的大友克洋印记，尤其展现了他作品中的创造性以及对机械、科技的迷恋。它集结了当时日本最顶尖的动画人：大友克洋、森本浩司、菅野洋子等。

《她的回忆》由森本浩司导演，值得一提的是我非常喜欢的导演今敏在其中担任了脚本和设定工作。今敏本身迷恋“虚实结合”的题材，着力于描写人物的记忆、梦境、臆想。在这部《她的回忆》和他另外一部代表作《红辣椒》中，都采用了梦境、回忆与现实“嵌套”的叙事模式。

作品讲述了未来太空被深度开掘，人类建立了完善的太空秩序。一艘太空运输船收到了神秘的求救讯号，SOS信号正是经典歌剧《蝴蝶夫人》，于是根据太空法决定对废墟进行搜救。登上信号源后，太空船员们遭遇了许多匪夷所思的诡异之事，小行星的主人——女高音歌唱家爱娃早已死去，却通过电脑模拟的投影将自己的记忆不断地延续了下去。她曾经亲手谋杀了自己出轨的恋人，认为只有这样才能让爱永恒。这个太空废墟遗存了爱娃的记忆，它的能量能够制造出仿真现实，而人类一旦进入这个现实，就会被自己的记忆所蛊惑，爱，或者内疚，每个心灵有所牵绊的人都会上演活生生的现实剧本，并最终走向毁灭。她的回忆魅惑了一名搜查的太空船员，另一名救援者则维持着清醒，不被自己沉重的记忆所吞噬。他们的对抗形成了巨大的漩涡，小行星最终于茫茫宇宙中炸为一朵盛放的玫瑰。结尾太空坟场这凄美的一幕正好映照了短片的英文片名《Magnetic Rose》。

2.

看完这部科幻短片之后，我对未来社会中的“记忆”产生了浓厚的兴趣。科幻电影中，既有直接将记忆和意识作为主题的电影，如《记忆裂缝》、《盗梦空间》、《记忆碎片》、《蝴蝶效应》、《红辣椒》；更常见的是将记忆相关的科幻设定加入到原本的主题中，丰富电影的内涵。

事实上，记忆在科幻电影中基本上没有任何特权地位，它不像时空穿越、人工智能、太空探索等科幻命题那么夺人眼球，它是平凡却又无处不在的。然而也要看到，在科幻电影中，记忆作为人类生存的基本主题和要素，成为了重新评估关于人类主体性的传统叙述的所在。

在现实生活中，人类的记忆是主观而易失的，遗忘是复杂的欲望和情感作出的必然选择。如果记忆可以从主观人格备份到数字化的外部媒介，永不遗忘的记忆是否会引发意想不到的问题？记忆来源于对客观事物的观察，反过来也影响着人类对客观世界的判断。如果记忆可以人为的检索、修复、传递甚至删除、篡改，那么真实和虚假的边界又该如何确认？……这些问题，科幻电影的创作者都代替我们进行了深入的探讨，接下来笔者就以《她的回忆》为主要案例，详细介绍一下科幻电影中对记忆的想象。

3.

科幻电影传达了未来派的幻想，私人的记忆可以通过技术的方式使得其透明和可见，例如将其投影为屏幕上的图像。个人记忆可以数字检索、展示、修复、转换或删除。

“记忆”已成为当代科幻电影中一个重要的主题，引发了社会对传统的主观人格概念的颠覆性思考。科幻电影最终试图将记忆从“主观心灵的内在感受”转换为“数字图像的外在媒介”。接下来我将以《她的回忆》为例，与大家分享科幻电影中经常出现的几种关于记忆的想象方式。

记忆的可视化

在满是废墟的太空坟场里，太空搜救者米格尔走到曾经繁华的宫殿尽头后，却看到一位美丽的红衣女子在碧草蓝天下惬意地散步，强风吹走了她的阳伞。这正是爱娃公主自己的回忆。影片中，爱娃的回忆片段多次在宫殿的不同角落与搜救者们不期而遇，过去与现实并不是两条不相交的平行线，两者会经常扭曲在一起，不分彼此。

我们之所以说这些画面不是简单的录像投影，而是女主角爱娃脑内的记忆，是因为它不仅重现了爱娃生前的场景和经历，更可以对搜救者们这些外来的入侵者们做出反应，与他们进行互动。在爱娃创造的这个记忆宫殿中，人们最在意的记忆被实物化了，即“仿真现实版的记忆”。片中有几个重要的伏笔，海因茨与米盖刚进入爱娃的宫殿时，海因茨不小心将一个小天使的塑像碰到，他立刻勃然变色；海茨茵独自闯入一个房间，旋转的八音盒从桌子上摔下，变得分崩离析，上面的形象也是一个可爱的小女孩。这些画面都指向了他内心最痛苦的梦魇——可爱的女儿因为自己的过失从屋顶坠下而死亡。而爱娃在最后为了彻底击垮他的意志，选择再一次在他面前重演了这一幕，海茨茵绝望地试图拥抱碰不到的女儿的“记忆体”，陷入了精神上的崩溃。

特效是科幻电影中重要的组成部分。为了更好地体现“记忆”这一虚幻的概念，电影常将之可视化成荧幕片段，而由于记忆本身的内容发生在观察记忆之前，因此常常会扰乱故事的线性叙事结构。在《她的回忆》中，记忆牵连了整个故事线索，甚至和观测者产生了适当的交互，从而推动整个故事情节的发展。

记忆片段的摘选也需要精心考虑。无意义的闪回会增大叙述的代价，甚至让观众产生不必要的期待。《她的回忆》尽量进行了留白，需要观众自行推测究竟发生了什么事情，利用较少的片面勾勒出了完整的历史情节。

记忆的编辑

本身人类的记忆有时就是虚构的，当人们遇到和自己的经验不一致的事情时，都会虚构自己的知觉和记忆，让一切看起来更合理——在爱娃的记忆中，她和她的恋人是恩爱的一对璧人，在她诱惑了米格尔作为恋人的替身之后，他们也表现得非常浓情蜜意。但是爱人却在婚礼前死掉了，在回忆这个痛苦的事件时，爱娃自觉地添加了一个“肇事者”，即她的狂热歌迷，却丝毫不觉自己的记忆出现了问题。事实上正是由于爱娃自己过于强大的控制欲和占有欲，不想面对爱侣的出轨，她选择了谋杀自己的未婚夫，让他最终且永远地只属于自己。

为了让事情更合理，我们往往会通过“脑补”的方式，用虚构来填补记忆间的空白。人们的记忆本身就是不完全可信的，如果还能被再次修改编辑，难以想象未来的世界会是怎样的混乱。

同时我们注意到，记忆与历史是不同的，还可能是矛盾和冲突的。历史需要权威来书写，而记忆则是每个人自己保存和拥有的。新闻资料的记载中，爱娃未婚夫的死因和爱娃的记忆一致，却和事实是不相符的。她对男朋友的暴力杀害由一种罪恶，冠冕堂皇地变成了“伟大的爱”。记忆不仅是由媒体收集和保存的，而且也由他们转载和代表。虽然这种媒体的这种力量已经被贬低，但人们仍然应该关注媒体是如何维持和保护记忆的。

数字化是否真的可靠？我们知道，记忆的外部持久化的同时也意味着它可以被人工篡改。“记忆编辑”主题的科幻作品就是讨论记忆的选择性遗忘和虚假记忆的植入。在惩治犯罪的过程中，良性的记忆编辑可以用来打消犯罪者的疑虑，而在记忆犯罪的主题中，恶意的篡改则会使角色信服一个预设的假象，并误导观众对正义和邪恶的初始判断，从而构造出矛盾的转折。

记忆植入与人工智能

在海茨因攻击了爱娃的形象之后，原来那个美丽的外壳破碎了，机器部件从女人的额头穿越而出。原来一切都是由于爱娃的执念，她在公众面前消失后，在太空中开始了与爱情回忆相伴的一生，将自己的记忆嫁接植入在了智能电脑上。一生都生活在荣誉光环中的她为死后的自己打造了一个光彩依旧的世界，在电脑模拟的全息投影中，她有关辉煌事业和美好爱情的记忆无尽延续。而爱娃在情感上的贪婪也借助智能电脑强大的能力，最终蛊惑了不知有多少人陷入这个玫瑰般美丽的死亡陷阱。

爱娃虽然杀死了自己的恋人，但本质上仍是一个普通人。但当智能机器被完全植入了她的记忆，爱娃残留下的恶意就会通过人工智能强大的能力，吞噬所有路过的生物，来填补爱娃生前的空虚和失落，造成的悲剧性后果是难以想象的。这也会引发一系列关于人工智能伦理的讨论：一个人的一生很大程度是由自己的每一段记忆构成和决定的，当这些记忆被植入机器人的新片中，机器人能否成为“真正的”人类？机器人做的事又与为它提供记忆的人类有多大的关系，谁来承担这个责任？

记忆植入类型电影的增多，侧面上也反映出工业界对软硬件的侧重转向。在前集成电路的时代，科幻电影中的机器人主要是以优秀的硬件支持为主，结实的金属外壳和人类无法达到的反应速度和机械力量似乎是机器人的标配，此时典型的“记忆”可以残存在假肢上。而在互联网和软件工程规模化的现在，科幻电影更加偏重对“智能”和“数字化”的讨论。记忆可以被数字化，可以被软件处理，可以移植到人工智能从而引发自我意识的觉醒。

4.

记忆数字化带来的益处

硅基芯片一直在模仿人类的行为，保存、检索文档都可以类比于碳基生物的记忆过程。然而不同的是，计算机技术中有着明确的容量边界，有着非易失的特性，可以被复制转存，甚至有着定向修改的可能性。

另一方面，人类的判断、决策很大程度上来源于自身的经验，而所谓的经验恰恰是经过筛选剔除后留下的记忆索引。人类自我意识的认同也很大程度上来源于自身的经历，而所谓的经历体现在个体上就是自身的记忆。二者的有机结合，给类似的科幻题材带来了无限的可能。

现实生活中，记忆通常被人们理所应当地当作事情的真相。个人的记忆虽然提供了一个“事实”的回放版本，但记忆在人脑不可靠的存储条件下会遗忘、扭曲，因而常常带有强烈的主观性。因此，科幻电影中人类带有缺陷的记忆和数字媒体可靠的非易失性存储之间常常会形成一个二元对立关系。在科技有能力将记忆持久化后，个人的记忆不再重要。计算机可以辅助消除可能出现的歧义，并使得探寻真相变得有迹可循。

同时，乐观的心理学家认为，人类一切心理困扰和创伤都是由过往经历所组成，黑暗和创伤性的记忆的破坏力也是非常强大的，而对记忆的消除可以解决这一切。《她的记忆》里太空坟场的最终秘密即是：毁灭一个人不需要凭借力量，而只需要凭借回忆。如果爱娃可以通过科技消除爱人出轨这一部分的痛苦回忆，那么遭到背叛后的一切悲剧将不会发生。

记忆数字化带来的技术难题和伦理隐患

数字化是否可以真实地再现记忆？

部分科幻电影试图把记忆当做是系统的内存，希望由人类对它进行全方位的控制。然而，上传、下载或操纵个人记忆的幻想，与人类思想实际工作的方式形成了鲜明对比。个人记忆是非常主观和经常不可靠的；在现实生活中，记忆和忘记往往超出了我们的控制，或受到太过人性化的影响（如选择性记忆或忘记）。

一些冷静的心理学者们也对未来的记忆编辑持消极态度，彼得·莱文就认为人类记忆系统是更加复杂的，并不是简单地由过往的经历组成。过往经历我们称为情境记忆，而情境记忆只是记忆系统中的一部分，而记忆系统中我们了解最少也最神秘的程序性记忆，我们依然知之甚少。这种程序性记忆可能反映着我们心理创伤形成的根本原因，因此，简单地消除或者扭曲记忆可能并无法真的让人们得到幸福，只会让人们在虚假的回忆里逃避并且迷失。

《她的回忆》的结局也展现了同样的态度，影片最重要也是最能体现主题的一句话：“记忆不是用来逃避的场所。”《她的回忆》像是航海时代水手们传说的海中女妖的宇宙版。女妖唱着你的耳朵无法拒绝的美妙歌曲，创造出难以抵御的脆弱幻想，吸引过往船只，吞噬着迷水手。动画中的宇宙背景或许暗指的是人脑。太空废墟保留了歌剧名伶爱娃的记忆，并把能量都用于制造出仿真的虚幻现实。人们受到了自己记忆的蛊惑，受到自己内心最牵挂事物的束缚。米格尔被爱娃的美貌所俘获，深陷与她的记忆间的纠缠，最后被无情地吞噬。清晰的回忆可能是痛苦的，但也是必须的。海茨茵对家有着深沉的眷恋，也因此差点受到了爱娃的蛊惑。在被记忆中活着的“女儿”拉住奔向虚假的幸福的时候，他踢到了自己的钱包——爱娃构建的虚幻世界中唯一保留的真实。看到照片上三张曾经幸福的脸，痛苦让海茨茵从虚幻中清醒了过来，挣扎着向爱娃开炮。不被记忆所蛊惑，人类才能走出人性的“太空深井”。

记忆是由复杂的欲望和情感决定的。爱娃对爱的强求，对记忆永存的执着，表现出了人类心底深处对孤独和改变的不安。然而不管是记忆还是爱情，都因其终将消逝才显得珍贵而美丽。她试图用一己之力征服时间，拒绝接受痛苦的回忆，坚持让一切停留在最好的瞬间，这本身也是人类过分贪婪的表现。最终的结局也必定是可悲可叹的。

5.

《她的回忆》是一个关于记忆的故事。在宇宙的尺度中，人们可以认真地思考，我们到底是应该选择被困于回忆中徘徊一生，还是面对现实走出阴霾。故事最终其实并没有给我们答案。米格尔深陷虚假的回忆，与制造回忆的智能电脑消失于宇宙深处。海茨因打破幻境找回了真实却痛苦的记忆，却没能逃过爆炸，人类不得已将坟场和海茨茵一起毁灭，他成为太空垃圾中的一员，在宇宙中进行无边的漂浮。

重新回到我们最初的遗问，如果记忆可以从主观人格备份到数字化的外部媒介，永不遗忘的记忆是否会引发意想不到的问题？如果记忆可以人为的检索、修复、传递甚至删除、篡改，那么真实和虚假的边界又该如何确认？……我们需要认识到，科学、技术和伦理之间的取舍正是科幻电影中的一个永恒的主题。科学的进步来源于人类的好奇心，技术的发展来源于对未来的期许，而伦理则来源于内心的恐惧和对自然的敬畏。科学引发技术的进步，而无限制的技术却有可能带来伦理上的困境，甚至导致灾难性的后果。
